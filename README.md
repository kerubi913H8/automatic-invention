# Automatic Invention - Nostalgic Tetris

端末上で楽しめる昔懐かしのテトリスを Python と curses で実装しました。

## 必要環境

- Python 3.10 以上
- UNIX 系端末 (macOS / Linux) など curses が利用できる環境

## 遊び方

```bash
python main.py
```

矢印キーとスペースキーで操作します。Vim 風の `h` / `j` / `k` / `l` キーにも対応しています。

| 操作 | 内容 |
| ---- | ---- |
| ← / → または `h` / `l` | ミノを左右に移動 |
| ↓ または `j` | ミノを 1 マス落下 |
| ↑ / `x` / `k` | 右回転 |
| `z` | 左回転 |
| スペース | ハードドロップ |
| `p` | 一時停止 |
| `r` | ゲームオーバー時にリスタート |
| `q` | 終了 |

ゲームはクラシックな 7 個ミノの袋ランダムとテトリス準拠のスコアリングを採用しています。ハードドロップ時には落下距離に応じたボーナスが加算されます。
