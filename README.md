# わくわくキッチン！🍳

子供向けクッキングゲーム - 楽しく料理を学ぼう！

## 🎮 ゲーム概要

「わくわくキッチン！」は3〜7歳の子供向けに開発された、インタラクティブな料理ゲームです。タッチ操作やマウス操作で、本物の料理を作る楽しさを体験できます。

## ✨ 主な機能

### 🐻 クッキーくまキャラクター
- 調理中に応援してくれるかわいいクマのキャラクター
- 表情が変化して、プレイヤーを励ましてくれる
- 完成した料理に対してリアクション

### 🍳 多彩なレシピ（6種類）
1. **オムレツ** - チュートリアル用の基本レシピ
2. **パンケーキ** - ふわふわ美味しい
3. **カレーライス** - 野菜を切って煮込もう
4. **ハンバーグ** - こねこねして焼こう
5. **いちごケーキ** - デコレーションが楽しい
6. **デコクッキー** - 型抜きとアイシング

### 📸 アルバム機能
- 完成した料理の写真を保存
- 最大50件までの料理記録
- 星評価、日付、レシピ名を記録

### 🎯 ゲームシステム
- **タップ**: 材料を入れる、卵を割る
- **ドラッグ**: 材料を移動、盛り付け
- **ミックス**: 円を描いて混ぜる
- **スワイプ**: 切る、焼く、巻く
- **タイミング**: 焼き加減を見極める
- **お絵描き**: ケチャップアートやデコレーション

### ⭐ アンロックシステム
- 星を集めて新しいレシピを解放
- オムレツ: 最初から利用可能
- パンケーキ: ⭐3個
- カレーライス: ⭐6個
- ハンバーグ: ⭐9個
- いちごケーキ: ⭐12個
- デコクッキー: ⭐15個

### 🎨 エフェクト＆サウンド
- キラキラエフェクト
- 湯気エフェクト
- ハートエフェクト
- 効果音（タップ、ジュージュー、注ぐ、割る等）
- 成功時のメロディ

## 🚀 遊び方

1. `index.html` をブラウザで開く
2. 「あそぶ！」ボタンをタップ
3. 作りたいレシピを選択
4. 画面の指示に従って調理
5. 完成した料理をアルバムに保存

## 💻 技術スタック

- **HTML5** - ゲーム構造
- **CSS3** - アニメーション＆スタイリング
- **Vanilla JavaScript** - ゲームロジック
- **Canvas API** - 調理画面の描画
- **Web Audio API** - 効果音生成
- **Local Storage** - データ保存

## 📱 対応デバイス

- PC（マウス操作）
- タブレット（タッチ操作）
- スマートフォン（タッチ操作）

レスポンシブデザインで、どんなデバイスでも快適にプレイできます。

## 🔧 開発情報

### ファイル構成
```
automatic-invention/
├── index.html      # メインHTML
├── game.js         # ゲームロジック
├── styles.css      # スタイル定義
└── README.md       # このファイル
```

### データ保存
ゲームは以下のデータをローカルストレージに保存します：
- `wakuwaku_stars`: 獲得した星の合計
- `wakuwaku_unlocked`: アンロックしたレシピ
- `wakuwaku_recipe_stars`: 各レシピの星評価
- `wakuwaku_album`: アルバムの料理記録
