# 三者面談予約・確定・通知アプリ

教師と保護者の2画面で、三者面談の希望提出〜自動割当〜確定通知まで行うWebアプリです。

## 使い方
1. `index.html` をブラウザで開く。
2. 教師画面（`#teacher`）で以下を実施:
   - 初期設定（期間、枠時間、休憩、締切）
   - 名簿登録（生徒名、保護者名、メール、兄弟姉妹グループ）
   - 保護者へ `#parent` URL を共有
3. 保護者画面（`#parent`）で生徒を選択し、第1〜第3希望を提出。
4. 教師画面で提出状況を確認し、自動マッチングを実行。
5. 必要なら手動調整して確定。メール通知（mailto）または印刷用通知を利用。

## 技術
- React（CDN）
- Tailwind CSS（CDN）
- localStorage
- ハッシュルーティング（`#teacher` / `#parent`）

## 注意事項
- データはブラウザ保存のため、削除すると消えます。
- ログイン認証はありません。共有URLの管理に注意してください。
